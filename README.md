# La Barberia - Precision Cuts · Old School Craft

Landing Page premium para una barbería de lujo industrial con estética vintage neoyorquina.

## 🎨 Estética y Diseño
- **Tema**: Lujo industrial oscuro.
- **Paleta de Colores**: 
  - Negro profundo (#0a0a0a)
  - Crema envejecida (#f5e6d3)
  - Dorado antiguo (#c9a84c)
  - Rojo bordó sutil (#8b1a1a)
- **Tipografía**: `Special Elite` unificada para toda la web (títulos, navegación y cuerpo), creando una estética de documento oficial mecanografiado que refuerza el aire nostálgico y artesanal.
- **Texturas**: Overlay de grano (grain) CSS puro para un aire cinematográfico.

## 🛠️ Características Técnicas
- **Single File**: Todo el proyecto (HTML, CSS y JS) se encuentra en `index.html`.
- **Responsive**: Diseño mobile-first adaptable a tablets y ordenadores.
- **Interacciones**:
  - Animaciones de carga escalonadas (staggered reveal) al hacer scroll.
  - Efecto Parallax suave en la sección Hero.
  - Modal de reserva funcional con validación y confirmación.
  - Navegación con scroll suave (smooth scroll).
- **Llamadas a la Acción (CTA)**: Botones interactivos con efectos de relleno dorado.

## 🚀 Cómo usar
Simplemente abre el archivo `index.html` en cualquier navegador moderno. No requiere dependencias externas ni servidores locales (excepto para cargar las fuentes de Google Fonts si tienes conexión).

## 📂 Directorio de Archivos
- `index.html`: Código fuente del sitio.
- `README.md`: Documentación del proyecto.
